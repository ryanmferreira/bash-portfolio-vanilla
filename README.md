Objetivo do Site:
Este site tem como objetivo apresentar o portfólio de um fotógrafo, tendo sido desenvolvido com base nos conhecimentos adquiridos durante as aulas de IHC.

Tecnologias utilizadas:
No desenvolvimento do site, foram utilizados HTML, CSS e JavaScript.

Decisões de design  relacionadas a IHC:
As principais decisões de design utilizadas foram relativas a Teoria das Cores, responsividade e experiência do usuário, visibilidade do status do sistema e consistência estrutural.
A paleta de cores e todo o site foram inspirados na identidade visual dos terminais Bash. Isso gareante também um alto contraste que reduz significativamente a fadiga visual durante longos períodos de uso.

Instruções de navegação:
Após o carregamento,o site iniciará na tela Home.Ao todo pode-se observar a existência de botões os quais levarão para a suas consequentes telas (Galeria e Sobre), as mesmas dispõem das seguintes informações:
Na tela de Galeria, é possível selecionar as imagens manualmente ou alterná-las utilizando os botões "Anterior" e "Próximo", que carregam as fotos junto com suas respectivas informações.
Consequentemente, a página "Sobre" dispõe apenas das informações sem interações em si.

No topo da página, um menu com os botões "Arquivo", "Sobre" e "Ajuda" auxilia na navegação pelo site. Na aba "Arquivo", existem as opções de sair do sistrema ou reiniciar a página (caso o usuário queira atualizar ou rever a animação de início). Já a aba "Sobre" mostra a "versão do sistema" junto do github dos dois participantes e o repositório do site. Por fim, a aba "Ajuda" mostra as informações sobre os atalhos de teclado, que facilitam a movimentação entre as páginas.
